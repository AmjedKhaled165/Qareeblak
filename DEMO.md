## 🎪 التأثيرات والحركات

### الويب
```
1. Scroll Reveal
   - العناصر تظهر تدريجياً عند التمرير

2. Gradient Text  
   - النصوص الهامة تتحرك بألوان زاهية

3. Blur In
   - العناصر تظهر بتأثير طمس

4. Floating Particles
   - جزيئات عائمة في الخلفية

5. Stagger Animation
   - عناصر متعددة تظهر بتسلسل
```

### الموبايل
```
1. Fade In
   - ظهور تدريجي للعناصر

2. Scale
   - تكبير/تصغير سلس

3. Slide
   - انزلاق من الأطراف

4. Bounce
   - حركة انزعاج جميلة

5. Rotate
   - دوران للعناصر البارزة
```
