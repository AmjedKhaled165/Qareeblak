# قريبلك - تطبيق Flutter

تطبيق موبايل احترافي متصل بنفس الويب (Next.js) مع نفس الديزاين والـ API.

## ✨ المميزات

- 🎨 **ديزاين حديث** - نفس الديزاين الجميل للويب
- 🔄 **متصل بنفس الـ API** - بيانات حقيقية مع الويب
- 💬 **نظام دردشة** - تواصل مباشر مع الخدمات
- 📍 **تتبع الموقع** - تتبع الطلبات في الوقت الفعلي
- 🌙 **Dark Mode** - دعم الوضع الليلي
- ⚡ **أداء عالي** - تطبيق سريع وسلس
- 🔐 **آمن** - Firebase Authentication

## 🚀 البدء

### المتطلبات
- Flutter 3.0+
- Dart 3.0+
- Android SDK / iOS SDK

### التثبيت

```bash
cd flutter_app
flutter pub get
```

### التشغيل

```bash
# على الويب (للاختبار)
flutter run -d chrome

# على الهاتف
flutter run

# على الـ iOS
flutter run -d ios

# على الـ Android
flutter run -d android
```

## 📱 الشاشات

1. **الرئيسية** - عرض الخدمات والفئات الشهيرة
2. **الاستكشاف** - البحث والفلترة على الخدمات
3. **الرسائل** - نظام الدردشة مع الخدمات
4. **الحساب** - إدارة الملف الشخصي والطلبات

## 🔧 البنية

```
flutter_app/
├── lib/
│   ├── main.dart              # نقطة البداية
│   ├── screens/               # الشاشات الرئيسية
│   ├── widgets/               # المكونات المعاد استخدامها
│   ├── services/              # خدمات الـ API
│   ├── providers/             # مزودي الحالة (Riverpod)
│   ├── config/                # الإعدادات
│   └── models/                # نماذج البيانات
├── pubspec.yaml               # الحزم والتبعيات
└── README.md
```

## 🔌 الاتصال بالـ API

التطبيق متصل بنفس الـ API من الويب:

```yaml
API Base URL: http://localhost:5000/api
WebSocket: http://localhost:5000
```

قم بتعديل `lib/config/api_config.dart` للإنتاج.
